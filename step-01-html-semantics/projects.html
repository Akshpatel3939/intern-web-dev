My Projects

Here is a selection of projects I've built during my internship and personal learning journey. Each one taught me something new about web development, problem-solving, and writing code for real users.

Portfolio Website

A personal portfolio website designed and built from scratch to showcase my web development skills. This project taught me how to apply semantic HTML5 structure, ensure keyboard navigability, and implement WCAG 2.1 accessibility guidelines throughout every page.

  • HTML5 — Semantic structure and landmark regions
  • CSS3 — Custom properties, Flexbox, and Grid layout
  • Accessibility — ARIA roles, skip links, focus management

To-Do App

A fully functional task manager built with vanilla JavaScript. Users can add, complete, and delete tasks, with all data persisted to the browser's localStorage so tasks survive page refreshes. This project deepened my understanding of the DOM, event handling, and browser storage APIs.

  • HTML5 — Semantic form elements and ARIA live regions
  • CSS3 — Transitions and state-based styling (done/not done)
  • JavaScript — ES6+, DOM API, localStorage, event delegation

Weather Dashboard

A real-time weather application that fetches live weather data for any city using the OpenWeatherMap API. The dashboard displays current temperature, humidity, wind speed, and a 5-day forecast. This project introduced me to asynchronous JavaScript, the Fetch API, JSON parsing, and handling API errors gracefully for users.

  • HTML5 — Semantic output and status regions
  • CSS3 — Responsive card layout with CSS Grid
  • JavaScript — Fetch API, async/await, JSON, error handling
  • OpenWeatherMap API — RESTful weather data service
