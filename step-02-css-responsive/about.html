About Me

Passionate developer, curious learner, and coffee enthusiast.

Who Am I?

Hi! I'm Alex Johnson, a second-year Computer Science student at State University with a deep passion for building things on the web. I started coding at 14 when I made my first "Hello World" page, and I've been hooked ever since.

I specialize in full-stack web development, with a particular love for creating clean user interfaces and robust APIs. When I'm not writing code, you'll find me reading tech blogs, contributing to open-source projects, or experimenting with new frameworks.

My Journey

My programming journey started with HTML and CSS in middle school, progressed through self-taught JavaScript, and then expanded into backend development with Node.js during my first year of university.

I've completed several online certifications including freeCodeCamp's Responsive Web Design, JavaScript Algorithms, and the Meta Front-End Developer Professional Certificate. I believe in continuous learning — the tech landscape changes fast, and staying curious is the best skill a developer can have.

What I Value

I care deeply about accessibility — building websites that everyone can use, regardless of ability. I also believe in performance: a fast website is a respectful website. And above all, I value clean, readable code that my future self (and teammates) will thank me for.
